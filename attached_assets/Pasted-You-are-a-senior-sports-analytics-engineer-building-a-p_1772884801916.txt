2. Projection Engine

Build a projection model using the following formula:

ProjectedScore =
(BaseStatProjection)
× RoleMultiplier
× MatchupMultiplier
× FormMultiplier
× TOGMultiplier

Role multipliers

InsideMid = 1.20
Wing = 1.05
HalfBack = 1.10
Forward = 0.85
KeyForward = 0.75

Matchup multiplier

OpponentPointsAllowedToPosition ÷ LeagueAverage

Form multiplier

Last5Average ÷ SeasonAverage

TOG multiplier

PlayerTOG ÷ LeagueAvgTOG

Store results in a player_projections table.

3. Breakout Detection Engine

Implement breakout scoring.

BreakoutScore =

(CBAIncrease × 0.40)
+
(TOGIncrease × 0.25)
+
(DisposalIncrease × 0.20)
+
(AgeFactor × 0.15)

AgeFactor:

22–25 years old = 1
Otherwise = 0.6

Flag players with breakout score above 0.65.

4. Fantasy Price Engine

Implement price movement model.

RollingAverage = (Last3Scores) ÷ 3

NewPrice = RollingAverage × MagicNumber

MagicNumber ≈ 5500

Breakeven =

((CurrentPrice / MagicNumber) × 3)
− Score1
− Score2

Display price projections for next 3 rounds.

5. Trade Optimiser

Create an algorithm that evaluates trades.

TradeGain =

(NewPlayerProjectedAvg − OldPlayerProjectedAvg)
× RemainingRounds

Rank all possible trades by projected points gained.

Allow user to input:

Current team
Available trades
Salary cap

Best two trades for the round.

6. Captain Optimiser

CaptainScore =

ProjectedScore
+
(CeilingProbability × CeilingScore)

CeilingProbability

Number of 130+ scores ÷ games played.

Return top 5 captain choices.
